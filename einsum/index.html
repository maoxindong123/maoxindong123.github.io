einsum爱因斯坦求和

2023/10/20 pytorch pytorch

参考博客

MoCo代码实现里有einsum，einsum叫爱因斯坦求和，可以用于向量内积，向量外积，矩阵乘法，转置和张量收缩等各种操作

a = torch.rand(2,3)
b = torch.rand(3,4)
c = torch.einsum("ik,kj->ij", [a, b])

einsum有2个参数，字符串和参数列表

字符串包括2种字符：

自由索引，出现在箭头右边的索引，比如上面的例子就是 i 和 j

求和索引，只出现在箭头左边的索引比如上面的例子就是 k

计算规则：

表达式会等价于1个外循环套1个内循环

外循环是按照右式的字符进行循环，例如例子中是ij，所以是下面

for i in range():
    for j in range():

内循环会沿着只出现在左式的字符进行求和（如果只有1个输入）or相乘求和（如果有\(\ge 2\)个输入）

sum=0
for k in range():
    sum+= a[i,k]*b[k,j]
c[i,j]=sum

求和的式子中，有几个输入就几个输入相乘，下标看左式（就是a[i,k]*b[k,j]）

求完和赋值给结果，结果的下标看右式（就是c[i,j]）

综合理解直接看第13个例子即可

13个einsum的例子：

  1.  b=torch.einsum('ii->i', [a])

 for i in range(i):
     for i in range(i):
         sum=0
         sum+=a[i,i]
         b[i]=sum

外循环沿i遍历，内循环无，所以是\(a[i,j]\)

结果是取出了a的对角线元素

  1. python torch.einsum('ij->ji', [a])

一眼转置

  1. python torch.einsum('...ij->...ji', [a])

高维张量的某些维度转置

  1. python torch.einsum('ij->', [a])

省略需要连->都省，这个没省，所以不是省略情况

外循环无，内循环沿i、j相加求和，所以是逐元素相加

  1. python torch.einsum('ij->j', [a])

外循环沿j，内循环沿i求和，所以是按行求和

  1. python torch.einsum('ik,k->i', [a, b])

矩阵乘法

  1. python torch.einsum('ik,kj->ij', [a, b])

还是矩阵乘法

  1. python torch.einsum('i,i->', [a, b])

i只在左边出现，或者说i是不同输入的重复维度，所以沿着i相乘求和，所以是向量内积

  1. python torch.einsum('ij,ij->', [a, b])

无外循环，内循环沿i、j遍历，会相乘并相加

最后结果就是a和b逐元素相乘后逐元素相加

  1. python torch.einsum('i,j->ij', [a, b])

外循环沿i、j遍历，无内循环，所以\(a[i]\ b[j]\)相乘

  1. python torch.einsum('ijk,ikl->ijl', [a, b])

矩阵乘法

  1. python torch.einsum('pqrs,tuqvr->pstuv', [a, b])

很高维，外循环沿着pstuv，内循环沿着qr进行相乘再求和
